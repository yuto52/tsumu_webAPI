# ツムツムコイン計算

## 経緯
ツムツムをやってて「コイン補正」の計算機が欲しくなったので作ってみた<br>
↓<br>
せっかくなので公開<br><br>

## 概要
- 主にツムツムの「コイン補正」を計算する
- コイン補正ごとの獲得コインも計算できるようにする予定
- ~~個人で使っていたものを公開しているだけなのでコードは汚いです~~

## 使い方（コイン補正計算）
1. サイトを開く
2. 消去数、消去前後のコイン枚数を入力
3. 計算ボタンを押す
 
下に赤文字で「ジャスト！」と出たら計算はほぼ確定で正しいです。<br>
出なかったら入力の際に見落としている部分があるかもしれません。<br>
&ensp; ex. 大ツム補正、他消去部分の見逃し、コイン数の見間違い
